## ⚡ Core Features

- **Dynamic Risk Scoring Engine** — An algorithmic, deeply parameterised scoring engine evaluating dynamic transaction volume, country-level sanctions, and PEP status natively within microseconds.
- **Enterprise Case Management** — High-fidelity task delegation, cross-department tracking modules, and comprehensive alert triangulation systems.
- **Regulatory Template Generator** — Produce fully automated, heavily formatted Word/Docx templates dynamically filled and parsed for instant regulatory submissions.
- **VIP Authentication Pass-through** — Fully scalable frontend router boundaries strictly locked down through dynamic JWT-bound layout components.
- **Periodic Event Pipelines** — Seamless integration models to manage long-term regulatory reviews without losing scope cache.

## 🛠 Technology Stack

This platform leverages an ultra-modern modular framework stack designed for extreme operational throughput and absolute UX persistence:

* **Frontend Framework:** React 18 
* **Compiler Engine:** Vite 6 with ultra-fast HMR bindings
* **CSS Architecture:** TailwindCSS v3.x coupled with native PostCSS integrations
* **Routing:** `react-router-dom` v6 Outlet-based structural routing
* **Data State/Caching:** `@tanstack/react-query`
* **Form Schemas:** `react-hook-form` resolved natively via `zod` parameter typing.

## 🚀 Getting Started

### 1. Installation Environment
Clone the repository and install the Vite architectural wrapper environment securely via your CLI:

```bash
git clone https://github.com/anjungowda/Regai.git
cd Regai
npm install
```

### 2. Live Node Execution
Spin up the internal server. Vite will safely compile the proxy mechanisms and trigger the development UI automatically on `http://localhost:5173`.

```bash
npm run dev
```
